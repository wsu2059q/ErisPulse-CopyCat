# CopyCat — 复读机模块
> 由 https://www.erisdev.com/#builder 模块构建器全权生成

检测群聊/私聊中连续重复消息，当同一消息被不同人连续发送 **2 次（含）以上**时自动复读。  
**同一条消息仅复读一次**，不会重复刷屏。

## 安装

```bash
epsdk install CopyCat
```

## 配置

在 ErisPulse 配置文件的 `[CopyCat]` 段中配置：

```toml
[CopyCat]
enabled = true           # 是否启用模块
trigger_count = 2        # 触发复读的连续重复次数
```

默认值：
| 配置项          | 默认值 | 说明                       |
|-----------------|--------|----------------------------|
| `enabled`       | true   | 是否启用模块               |
| `trigger_count` | 2      | 触发复读的连续重复次数     |

## 使用

无需任何命令，模块自动运行。

### 示例

```
用户A: 今天天气真好
用户B: 今天天气真好        ← 不同人重复第 2 次
机器人: 今天天气真好        ← 自动复读（仅这一次）
用户C: 今天天气真好        ← 已复读过，不再复读
```

### 行为说明

- **不同人连续发送**：只有不同用户连续发送相同消息才会计数
- **同一用户多次发送**：不算复读，不计数
- **中间插入其他消息**：重置计数，重新开始
- **已复读的消息**：不再复读

## 卸载

```bash
epsdk uninstall CopyCat
```
